# pifm
Analysis of MundeepL's PiFM (https://github.com/MundeepL/PiFM) was done in this project. Data transfer using IR and Raspberry Pi was also implemented

This repository contains reports relevant to:

- Mathematical + Physical analysis of the pifm
- Hardware analysis of the pifm
- Software analysis of the pifm
- Software and hardware analysis of RTL-SDR
- Text and file transfer using IR
